# ChatGPT-FairXAI
"GPT-FairXAI" investigates the potential of Large-scale Language Models (LLMs) like OpenAI's GPT in the domain of credit risk assessment. Our findings suggest that when judiciously designed prompts and domain-specific knowledge are integrated, LLMs can achieve parallel performance to traditional ML modelsâ€”while requiring significantly less data.
